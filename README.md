# Kütüphanecim

Kişisel kütüphane takip uygulaması.

## Özellikler
- Kitap ekleme, düzenleme, silme
- Raf ve liste görünümü
- Arama ve filtreleme (tür, dönem, durum, seri)
- Okuma durumu takibi (okunacak / okunuyor / okundu)
- Puan ve not sistemi
- İstatistik ve grafikler
- Rozet sistemi
- Renkli kapaklar ve seri rozetleri

## Kullanım
Direkt tarayıcıda açın — hesap gerekmez, veriler tarayıcınızda saklanır.
